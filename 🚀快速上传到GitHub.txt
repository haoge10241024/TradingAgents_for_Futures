【推荐方法1：使用一键脚本（最简单）】

第1步：在GitHub创建仓库
  1. 访问 https://github.com/
  2. 点击右上角 "+" → "New repository"
  3. 输入仓库名（如：TradingAgents-Futures-AI）
  4. 选择Public或Private
  5. 不要勾选 "Add a README file"
  6. 点击 "Create repository"
  7. 复制仓库地址（如：https://github.com/用户名/仓库名.git）

第2步：双击运行脚本
  1. 双击 "一键上传脚本.bat"
  2. 粘贴刚才复制的仓库地址
  3. 按回车

第3步：输入GitHub凭据
  用户名：你的GitHub用户名
  密码：Personal Access Token（不是登录密码！）
  
  如何获取Token：
  1. GitHub → Settings → Developer settings
  2. Personal access tokens → Tokens (classic)
  3. Generate new token → 勾选 "repo" 权限
  4. 复制生成的token（ghp_开头）

完成！所有文件和文件夹已上传！

============================================================

【推荐方法2：GitHub Desktop（最直观）】

第1步：下载安装 GitHub Desktop
  下载：https://desktop.github.com/

第2步：打开项目
  1. 打开GitHub Desktop
  2. File → Add local repository
  3. 选择 TradingAgents_for_Futures 文件夹
  4. 点击 "create a repository"

第3步：发布到GitHub
  1. 输入提交信息：Initial commit
  2. 点击 "Commit to main"
  3. 点击 "Publish repository"
  4. 完成！
